# Project Name: 
KeenKeeper

# Description:
KeenKeeper is a platform to be connected with the friends.

# Technology I used on this Project:
1. React JS
2. React Router
3. Tailwind CSS and Daisy UI
4. Recharts for chart

# Features of my project:

When a user click on a friend card, it will redirect the user to details page of that friend. In this page there are three buttons such as- Call, Text and Video. When the user click on any of the btn it will show a toast and data will be added to timeline page. In timeline page there will be all data of click and a dropdown to sort specific item. And a stats page to show the timeline data visually by pie chart from Recharts.
